# JavaFX Calculator

Ứng dụng calculator đơn giản viết bằng JavaFX (Maven project).

## Yêu cầu

- Java 17 trở lên (JDK)
- Maven

## Cách chạy

Trong thư mục project này, mở terminal và chạy:

```bash
mvn clean javafx:run
```

Nếu bạn dùng IntelliJ IDEA hoặc một IDE khác:

- Mở project Maven này.
- Đảm bảo IDE dùng JDK 17+.
- Chạy lớp `com.example.calculator.CalculatorApp` làm ứng dụng JavaFX.
